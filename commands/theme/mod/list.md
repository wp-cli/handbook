# wp theme mod list

Gets a list of theme mods.

### OPTIONS

[\--field=&lt;field&gt;]
: Returns the value of a single field.

[\--format=&lt;format&gt;]
: Render output in a particular format.
\---
default: table
options:
  - table
  - json
  - csv
  - yaml
\---

### EXAMPLES

    # Gets a list of theme mods.
    $ wp theme mod list
    +------------------+---------+
    | key              | value   |
    +------------------+---------+
    | background_color | dd3333  |
    | link_color       | #dd9933 |
    | main_text_color  | #8224e3 |
    +------------------+---------+
