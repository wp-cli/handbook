`wp role list` - List all roles.

### OPTIONS

[--fields=&lt;fields&gt;]
: Limit the output to specific object fields. Defaults to name,role.

[--format=&lt;format&gt;]
: Output list as table, CSV or JSON. Defaults to table.

### EXAMPLES

    wp role list --fields=role --format=csv
