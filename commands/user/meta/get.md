# wp user meta get

Get meta field value.

### OPTIONS

&lt;user&gt;
: The user login, user email, or user ID of the user to get metadata for.

&lt;key&gt;
: The metadata key.

[\--format=&lt;format&gt;]
: Render output in a particular format.
\---
default: table
options:
  - table
  - csv
  - json
  - yaml
\---

### EXAMPLES

    # Get user meta
    $ wp user meta get 123 bio
    Mary is an WordPress developer.
